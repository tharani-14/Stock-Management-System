<h1 >Stock management System</h1>

<h3>Abstract</h3>
<p>The Stock Management System Project focuses mainly on the management of stock inventories. Also, the system displays all the lists of suppliers too. In addition, the system allows managing departments too. This project is divided into two categories: Admin and Other Departments. In an overview of this web application, the staff can simply view their allocated stock items. For this, an admin needs to allocate stock items in order to view them under the staffâ€™s account. Besides, the staff can view reports for their allocated stock items and update their profiles.</p>
